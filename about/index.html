项目经验

1. uemfit(在线约健身教练O2O平台)

公众号：umefit优美健身服务号
2016.4-2016.5

umefit是一个预约健身教练的O2O平台，整个产品分为三个部分，顾客端、教练端和网站运营后台，顾客端和教练端是在微信端，网站运营后台在PC端。
在项目中我负责前端部分的全部开发，构建项目架构和自动化流程，与后端进行数据交互，完成所有的view组件。
项目的技术主要使用 es6 + vue + vue-router + vue-resource + webpack + sass，整个项目是一个单页的webapp，采用组件化的开发模式，前端控制路由，同时分离data层和view层，通过 RESTFul API 和后端交互，前后端分离，通过约定api配合开发。在移动端，特别是微信内置浏览器解决比如上传文件等bug与兼容，在移动端的各种不同分辨率屏幕上做了自适应，实现了友好的用户体验。
